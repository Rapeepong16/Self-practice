ข้อ 1 
-การใช้วัตถุ เป็นตัวเก็บข้อมูลแบบ key-value pair
-การloop เพื่อเข้าถึงและประมวลผลข้อมูลใน array ด้วย for loop
-เราใช้ frequencyNum เป็น Object เพื่อเก็บว่าตัวเลขซ้ำกี่ครั้ง เมื่อเจอเลขหนึ่งตัวในอาร์เรย์ เราจะเช็คว่าเลขนั้นมีใน frequency หรือยัง ถ้ามีแล้ว ก็เพิ่มค่า count ทีละ 1 ถ้าไม่เคยเจอมาก่อน ก็สร้าง key ใหม่และตั้งค่าเป็น 1 ใช้ for loop เพื่อวนผ่านทุกตัวใน array

ข้อ 2
-การใช้ nested loops เพื่อเปรียบเทียบค่าภายใน array
-การใช้ตัวแปร Boolean เป็นตัวบ่งชี้สถานะ 
-วน loop i เพื่อดูแต่ละค่าที่อยู่ใน array ต้นฉบับ ในloop j วนเช็คว่า ค่าที่เรากำลังดูนั้นเคยถูกเก็บไว้ในอาร์เรย์ uniqueArr หรือยัง ถ้าเจอว่ามีค่าเดียวกันแล้ว เราจะตั้ง flag exists = true เพื่อบอกว่าไม่ต้องเพิ่มค่าเดิมซ้ำ ถ้า exists ยังเป็น false (แปลว่ายังไม่เจอค่าเหมือนกันใน array ใหม่) ก็เพิ่มค่าลงใน uniqueArr สุดท้ายจะได้อาร์เรย์ใหม่ที่ไม่มีค่าซ้ำเลย
